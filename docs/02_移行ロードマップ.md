## 1. セットアップシェル（`src/scripts/`）の追加候補

## 4. 追加時のチェックリスト

新しい要素を入れるときに、忘れず連動更新するもの。

- セットアップシェル追加時:
  - [ ] [`03_実装カタログ.md §2`](./03_実装カタログ.md#2-srcscripts-シェル群) に行を追加
  - [ ] [`04_検証.md`](./04_検証.md) に動作確認手順を追加
  - [ ] 既存シェルと同じ構造（`set -euo pipefail` / `require_*` / 冪等 / `print_next_steps`）に揃える
- プロンプトメモ追加時:
  - [ ] `src/{pg,office-task}/README.md` に `## プロンプトN: <概要>` 形式で追記（共通項目を DRY 化しないこと）
  - [ ] [`03_実装カタログ.md §3`](./03_実装カタログ.md#3-srcpg--srcoffice-task-プロンプト集) のプロンプト目録に行を追加
  - [ ] サンプルデータが必要なら `src/<area>/data/` に配置
- ドキュメント追加時:
  - [ ] [`docs/README.md`](./README.md) の一覧に行を追加
