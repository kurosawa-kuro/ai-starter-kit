# 00. クイックスタート（15 分で AI を動かす）

～ Windows ユーザーが最短で **Claude Code を起動するまで** の超短縮ルート ～

> 💡 このドキュメントは **「動かすこと最優先」** で書いています。
> 仕組みの説明や詳細手順は [`01_環境構築ガイド.md`](./01_環境構築ガイド.md) / [`02_ハンズオン.md`](./02_ハンズオン.md) を参照。
> エラーが出たら → [`03_WSL操作.md`](./03_WSL操作.md) と [`../04_検証.md`](../04_検証.md)。

## 全体の流れ

```
   ⏱ 15〜20 分（PC スペックとネットワークによる）

   ① WSL 導入（PowerShell 管理者）  …… 5 分 + 再起動
       ↓
   ② VSCode + WSL 拡張             …… 3 分
       ↓
   ③ git clone + setup-*.sh        …… 5 分
       ↓
   ④ ターミナル開き直し → claude    …… 2 分

   → 🎉 AI が話しかけてくれる状態
```

---

## ① WSL を入れる（PowerShell 管理者で実行）

スタートを **右クリック** → 「**ターミナル（管理者）**」を選択。
> PowerShell が分からない場合は [`01_環境構築ガイド.md`](./01_環境構築ガイド.md) の「📌 補足: PowerShell とは」を先に。

```powershell
wsl --install
```

   - Ubuntu が自動で入る
   - 完了後 **PC を再起動**
   - 再起動後 Ubuntu の黒い画面が出るので **ユーザー名（英小文字）とパスワード** を設定

確認:
```powershell
wsl -l -v
```
→ `Ubuntu` が `Running` か `Stopped` で表示されれば OK。

---

## ② VSCode + WSL 拡張を入れる

   1. https://code.visualstudio.com/ から「Download for Windows」→ インストーラ実行
   2. VSCode を起動 → 左サイドバー「拡張機能」→ 検索欄に `WSL` → 発行者 **Microsoft** の "WSL" を **Install**
   3. 左下の **緑「><」** をクリック → "**Connect to WSL**"
   4. 左下が **`WSL: Ubuntu`** になれば接続成功

---

## ③ clone してセットアップ

VSCode 上部メニュー「ターミナル」→「新しいターミナル」（または `Ctrl + @`）。
**左下が `WSL: Ubuntu` であることを確認** してから:

```bash
# 作業フォルダを用意
mkdir -p ~/dev && cd ~/dev

# 本キットを取得
git clone https://github.com/kurosawa-kuro/ai-starter-kit.git
cd ai-starter-kit

# 必須セットアップ（合計 3〜5 分程度）
bash src/scripts/setup-python.sh        # Python 3
bash src/scripts/setup-claude-code.sh   # Claude Code（nvm + Node.js + CLI）
```

> ⚠️ `setup-claude-code.sh` 完了後は **必ず一度ターミナルを閉じて開き直して** ください（`nvm` の PATH 反映のため）。

---

## ④ Claude Code を起動

ターミナルを開き直したら:

```bash
cd ~/dev/ai-starter-kit
claude
```

   - 初回はブラウザが開いて Anthropic アカウントの認証を求められる → サインイン
   - プロンプトが表示されたら成功 🎉

試しに何か聞いてみる:
```
こんにちは。このリポジトリの src/pg/README.md にある
プロンプトを使って、偶数判定の Python CLI を作って。
```
