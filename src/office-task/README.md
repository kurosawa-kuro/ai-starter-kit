## プロンプト5: 議事録テンプレ生成

- **依頼内容**: 以下の情報から Markdown 形式の議事録ファイルを作成
- **入力情報**:
  - 議題（タイトル）
  - 日時（YYYY-MM-DD HH:MM）
  - 場所（任意）
  - 参加者（カンマ区切り）
  - 主な議論項目（箇条書き）
- **出力フォーマット**:
  ```markdown
  # 議事録: <議題>

  ## 日時
  ## 場所
  ## 参加者
  ## アジェンダ
  ## 議論内容
  ## 決定事項
  ## アクションアイテム（担当 / 期限）
  ## 次回予定
  ```
- **保存先**: 本リポジトリの `src/office-task/data` に `<YYYY-MM-DD>_<議題>.md` として保存
- **回答**: 日本語で、日本人にわかりやすい説明
